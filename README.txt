        จาการเรียนรู้ HTML โดยลองทำ Lab 1 ทำให้เข้าใจโครงสร้างของเว็บเพจมากขึ้น รู้จักการใช้แท็กพื้นฐาน และ Semantic HTML ให้เหมาะสมกับเนื้อหาและการจัดลำดับความสำคัญ จนสามารถสร้างเว็บเพจ 1 หน้าได้อย่างเป็นระเบียบและอ่านง่าย ทำให้เข้าใจการเชื่อมโยงเนื้อหาภายในและภายนอกหน้าเว็บด้วยลิงก์ต่างๆ สุดท้ายนี้การเรียนรู้ HTML ด้วยตนเองทำให้ได้ฝึกค้นคว้าและแก้ปัญหาด้วยตัวเอง ช่วยให้จำและเข้าใจได้ดีกว่าเดิม
